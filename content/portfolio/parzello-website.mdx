# Studi Kasus: Parzello Website

**Parzello** adalah sebuah agensi digital kreatif yang berfokus pada penyediaan solusi teknologi modern, desain UI/UX kelas dunia, serta pengembangan aplikasi mobile dan web yang inovatif. Website ini dibangun sebagai wajah digital agensi guna mempresentasikan portofolio, layanan, dan keahlian tim kepada calon klien global.

## 🚀 Fitur Utama
1. **Desain Interaktif Berbasis Animasi:** Memanfaatkan library animasi tingkat tinggi untuk memberikan transisi halus dan interaksi pengguna yang menarik tanpa mengurangi performa.
2. **Portofolio Showcase Dinamis:** Menampilkan katalog proyek terbaik agensi dengan filter berdasarkan kategori (Desain UI/UX, Web, dan Mobile).
3. **Formulir Kontak Terintegrasi:** Memungkinkan calon klien untuk langsung berkonsultasi mengenai proyek mereka dengan pengiriman notifikasi instan ke tim internal.
4. **Desain Responsif & Kinerja Premium:** Optimal diakses di semua resolusi perangkat (smartphone, tablet, maupun layar desktop 4K).

## 🛠️ Teknologi yang Digunakan
* **Frontend Framework:** Next.js (App Router) & React untuk performa pemuatan cepat berbasis Static Site Generation (SSG).
* **Styling & UI:** TailwindCSS & HeroUI (NextUI) untuk komponen UI yang modular dan konsisten.
* **Animasi:** Framer Motion untuk micro-animations dan transisi antar halaman.
* **Hosting:** Vercel untuk deployment otomatis dengan integrasi Continuous Integration / Continuous Deployment (CI/CD).

## 🧠 Tantangan & Solusi
* **Tantangan:** Mengintegrasikan banyak efek animasi interaktif tingkat tinggi (seperti parallax dan scroll-based animations) berpotensi menurunkan skor performa (Lighthouse) website secara drastis di perangkat berspesifikasi rendah.
* **Solusi:** Menerapkan teknik *lazy loading* untuk komponen non-kritis, mengompresi aset gambar ke dalam format WebP modern, serta menggunakan *hardware-accelerated animations* di CSS/Framer Motion untuk memastikan rendering grafis optimal dan mulus di 60 FPS.
